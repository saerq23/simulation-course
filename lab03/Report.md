# Отчет

## Цель
Целью работы является изучение принципов имитационного моделирования и клеточных автоматов, а также разработка программной модели распространения лесного пожара с возможностью визуализации процесса.

## Теория
Клеточный автомат — это математическая модель, состоящая из решётки клеток, каждая из которых может находиться в одном из конечного числа состояний.

Состояние каждой клетки изменяется во времени в соответствии с определёнными правилами, которые зависят от состояния соседних клеток.

Основные компоненты клеточного автомата:
Решётка клеток — пространство модели (двумерная сетка).
Набор состояний клеток.
Окрестность клетки (набор соседей).
Правила перехода между состояниями.

В данной работе используется двумерная решётка клеток и окрестность Мура, включающая восемь соседних клеток.

## Описание модели
Модель представляет собой двумерную сетку клеток, каждая из которых может находиться в одном из следующих состояний:
| Состояние | Описание         |
| --------- | ---------------- |
| EMPTY     | пустая клетка    |
| TREE      | клетка с деревом |
| BURNING   | горящее дерево   |
| ASH       | сгоревшее дерево |

Основные правила модели:

Если клетка содержит дерево и рядом есть горящая клетка, то дерево может загореться.

Горящее дерево на следующем шаге превращается в пепел.

Пепел со временем становится пустой клеткой.

На пустой клетке может вырасти новое дерево.

Дерево может загореться случайно (например, от молнии).

В модели реализованы дополнительные факторы, влияющие на распространение пожара:

1. Ветер

Ветер увеличивает вероятность распространения огня в определённом направлении.
Пользователь может выбрать направление и силу ветра.

2. Влажность

Повышенная влажность уменьшает вероятность возгорания деревьев.

3. Различные типы леса

В модели используются разные типы леса:
обычный лес;
плотный лес (горит быстрее);
влажный лес (горит хуже).

Эти типы имеют различную вероятность воспламенения.

## Вывод

В ходе выполнения работы были изучены принципы клеточных автоматов и имитационного моделирования.

Была разработана программная модель распространения лесного пожара с графической визуализацией.

Реализованные дополнительные правила (ветер, влажность и различные типы леса) позволяют более реалистично моделировать процесс распространения огня и анализировать влияние различных факторов на поведение системы.
